GitHub has a 50 MB file upload limit so the case file was split into two files.  You will need to combine the two files into a single file using the following command while in this folder.

cat forensics.E01a* > forensics.E01

Then check the MD5 hash of forensics.E01 to match: 6e1d72d2798e6f7cdc3876544d493aed
